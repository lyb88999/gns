API 参考文档

Endpoint: http://<your-server>:8080
认证方式: Bearer Token

1. 认证 (Auth)

1.1 用户登录

POST /api/v1/auth/login 
{
  "username": "admin",
  "password": "password"
}

1.2 获取当前用户

GET /api/v1/auth/me

2. 通知 (Notify)

2.1 发送通知

POST /api/v1/notify

发送单条通知。支持附件与自定义变量。

{
  "taskId": "UUID...",
  "data": { "key": "value" },
  "priority": "High"
}

3. 任务管理 (Tasks)

3.1 创建任务

POST /api/v1/tasks 
{
  "name": "每日报表",
  "triggerType": "cron",
  "cronExpression": "0 0 12 * * ?",
  "channels": ["Email", "DingTalk"],
  "messageTemplate": "模版内容..."
}

3.2 任务列表

GET /api/v1/tasks?page=0&size=10

4. 日志 (Logs)

GET /api/v1/logs

参数: page, size, status (SUCCESS/FAILED/BLOCKED), search
