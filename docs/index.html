核心优势

为稳定性而生

不仅仅是发送消息，更是对每一条触达的精准控制。

高并发处理

基于 Redis Stream 的异步队列设计，轻松应对每秒万级消息吞吐，确保高峰期系统不崩。

企业级安全

内置 RBAC 权限体系，支持团队隔离。API 访问采用 Token 认证，支持 IP 白名单与审计日志。

灵活流控

支持按任务设置频控（如每小时 100 条），并支持夜间静默模式，避免打扰用户。

开发者优先

多语言 SDK 支持

无论使用哪种语言，这几行代码都能让你的应用拥有强大的通知能力。

example.py 
# 安装: pip install gns-sdk

from gns_sdk import GNSClient

client = GNSClient(
    base_url="https://api.gns.com",
    token="sk_live_..."
)

client.send_notification(
    task_id="order-shipped",
    data={"order_id": "1001"},
    priority="high"
)
// 安装: npm install gns-sdk

const GNSClient = require('gns-sdk');

const client = new GNSClient(
    'https://api.gns.com', 
    'sk_live_...'
);

await client.sendNotification({
    taskId: 'order-shipped',
    data: { orderId: '1001' },
    priority: 'high'
});
// 安装: go get github.com/gns/sdk/go

import "github.com/gns/sdk/go/gns"

client := gns.NewClient(
    "https://api.gns.com", 
    "sk_live_...",
)

client.SendNotification(gns.SendRequest{
    TaskId: "order-shipped",
    Data: map[string]interface{}{
        "order_id": "1001",
    },
})
// Maven Dependency

GnsClient client = new GnsClient(
    "https://api.gns.com", 
    "sk_live_..."
);

SendRequest req = SendRequest.builder()
    .taskId("order-shipped")
    .data(Map.of("order_id", "1001"))
    .build();

client.sendNotification(req);

集成步骤

  • 1

    获取 API Token

    登录控制台，进入「访问控制」->「API Token」，创建一个新的令牌。

  • 2

    创建通知模版

    在「任务管理」中新建任务，定义发送渠道和模版内容（支持 ${var} 变量）。

  • 3

    集成 SDK

    参考上方的 SDK 示例代码，将通知集成到您的业务系统中。
